How It Works
On-Call System
  • Primary: First person notified for new support requests
  • Backup: Notified if primary doesn't respond within timeout
  • Escalation: Automatic escalation through backup team members
Notification Flow
  1. Support request received via Telegram bot
  2. System identifies current on-call person
  3. Primary on-call person notified via Telegram
  4. If no response in 15 minutes, backup is notified
  5. Process continues until someone responds
